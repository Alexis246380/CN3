\# Projet AlgoBox – TP GitHub



\## Description

Projet réalisé dans le cadre d’un TP visant à utiliser GitHub comme outil collaboratif avec AlgoBox.



\## Prérequis

\- AlgoBox installé

\- Git installé



\## Exécution

1\. Cloner le dépôt GitHub

2\. Ouvrir le fichier `.alg` présent dans le dossier `src/` avec AlgoBox

3\. Exécuter l’algorithme depuis AlgoBox
